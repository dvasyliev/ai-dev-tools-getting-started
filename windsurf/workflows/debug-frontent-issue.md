# Debug Frontend Issue

## Goal

Identify and fix a frontend bug in the project.

## Steps

1. Understand the bug description.
2. Locate relevant files and components.
3. Identify the root cause.
4. Suggest a fix and apply it.
5. Write a short summary using the template below.

## Final Summary Template

```txt
Debug Summary

- Issue: [short description of the bug]
- Root cause: [what caused the issue]
- Files checked: [list of relevant files]
- Fix applied: [what was changed]
- Status: [fixed / not fixed / needs follow-up]
- Notes: [optional extra notes]
```
