# CLAUDE.md

This file configures Claude Code behavior for this project.

## Rules

The following rules are automatically applied during all work in this project:

- [Coding Style](./rules/coding-style.md) — immutability, KISS/DRY/YAGNI, naming conventions, file organization
- [Security](./rules/security.md) — secret management, input validation, SQL injection, XSS prevention
- [Code Review](./rules/code-review.md) — review checklist, severity levels, approval criteria
- [Git Workflow](./rules/git-workflow.md) — conventional commits, PR workflow
- [Performance](./rules/performance.md) — model selection strategy, context window management, Plan Mode usage
