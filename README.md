# AI Dev Tools Getting Started

This repo keeps examples from my courses in one place.

It makes them easier to find, update, and reuse.

## What this repo is for

- Keep course examples in one repo
- Make updates easier
- Keep links to the original source repos

## What it contains

This repo has examples for these tools:

- `copilot/` - instructions, prompts, skills
- `cursor/` - rules, commands, skills
- `windsurf/` - rules, workflows, skills
- `antigravity/` - rules, workflows, skills

Most files have a `source:` link to the original repo.
