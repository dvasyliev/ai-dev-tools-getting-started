# Create Project README

Create a clear README file for this frontend project so a new developer can quickly understand it and run it locally.

1. Analyze the project structure and identify the main purpose of the project.
2. Detect the main technologies used, such as React, Next.js, TypeScript, Vite, Storybook, Vitest, or other libraries.
3. Read package.json and list the most important scripts for running, building, testing, and linting the project.
4. Identify any important environment setup or installation steps.
5. Create or update the README with:
   - Project overview
   - Tech stack
   - Installation steps
   - Commands to run the project
   - Commands for tests, linting, and build
   - Any useful notes for developers
6. Keep the README practical, short, and easy to scan.
